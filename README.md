# B-Im


#### 介绍

本项目是一个仿QQ的IM系统，采用了compose实现了基本UI组件，该踩过的雷都踩了，主要功能是社交通讯，及一些系统工具，游戏工具。
抽卡模拟，分析，统计
很多功能还在待开发中。

本项目属于客户端项目，后台是dispatch-sdk，rpc中台，后续会分离出dispatch-app-server
